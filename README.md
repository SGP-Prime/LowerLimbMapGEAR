# LowerLimbMapGEAR

Mapa interativo do **plexo lombossagrado** e dos bloqueios ecoguiados do membro inferior — Sessão 2 das Aulas de Internos do **GEAR** (Grupo de Ecografia e Anestesia Regional).

Companheiro da Lição 1 ([UpperLimbMapGEAR](https://github.com/SGP-Prime/UpperLimbMapGEAR)).

## Conteúdo

- `index.html` — mapa interativo autossuficiente (sem dependências externas): pan/zoom, 25 bloqueios alternáveis, predefinições por região e gaveta para telemóvel.
- `.nojekyll` — serve os ficheiros tal como estão no GitHub Pages.

## Publicação (GitHub Pages)

1. Criar o repositório `LowerLimbMapGEAR` na conta **SGP-Prime**.
2. `Settings → Pages → Build and deployment → Source: Deploy from a branch → main / root`.
3. Após o push, fica disponível em:
   `https://sgp-prime.github.io/LowerLimbMapGEAR/`

## 25 bloqueios cobertos

Parede abdominal (TAP subcostal/lateral/posterior, bainha do recto, ilioinguinal–ílio-hipogástrico),
quadrado lombar (QL1/QL2/QL3), ESP lombar, plexo lombar (Shamrock), PENG,
femoral, fáscia ilíaca supra/infrainguinal, canal dos adutores,
cutâneo femoral lateral, obturador, cutâneo femoral posterior,
isquiático (subglúteo, anterior, poplíteo), iPACK, pudendo, safeno, bloqueio do tornozelo.
